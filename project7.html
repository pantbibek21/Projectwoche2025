Write the Web

About

Der Workshop „Write the Web“ bietet die Möglichkeit, die Grundlagen des Webdesigns kennenzulernen und praktisch umzusetzen. Ganz ohne Vorkenntnisse starten die Teilnehmenden und lernen Schritt für Schritt den Umgang mit HTML, CSS und JavaScript. Ziel ist es, am Ende eine eigene Website zu erstellen, die die Projektwoche dokumentiert. Im Verlauf des Workshops wird gezeigt, wie eine Webseite aufgebaut ist und welche Rolle die verschiedenen Sprachen spielen: HTML strukturiert die Inhalte, CSS sorgt für ein ansprechendes Design und JavaScript bringt Interaktivität hinein. So erwerben die Schülerinnen und Schüler nicht nur theoretisches Wissen, sondern auch praktische Fähigkeiten, die sie direkt anwenden können. Besonders spannend ist, dass mit jedem Schritt sofort Ergebnisse sichtbar werden. Schon nach kurzer Zeit lassen sich Texte, Bilder oder Farben auf einer Seite verändern. Mit etwas mehr Übung können eigene Layouts gestaltet oder interaktive Elemente hinzugefügt werden. Die Arbeit an der Website fördert Kreativität, Problemlösekompetenz und technisches Verständnis gleichermaßen. Am Ende hält jede Gruppe eine funktionierende Website in den Händen – ein sichtbares Ergebnis ihrer eigenen Lernarbeit.
